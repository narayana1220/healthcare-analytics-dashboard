# HealthStat | Healthcare Analytics Dashboard

## Project Overview
This Power BI dashboard analyzes elective hip replacement inpatient stays across New York State hospitals.

## Features
- Length of Stay (LOS) Analysis
- Cost Comparison
- Hospital Profiling
- Key Influencer Analysis
- Interactive Filters

## Tools Used
- Power BI
- DAX
- Power Query
- Data Modeling

## Key Insights
- Average LOS: 2.65 days
- Average Cost per Discharge: $21K
- Mortality risk impacts LOS significantly
